## ğŸ¨ Customization Guide

### 1. Personal Information

Update the following sections in `index.html`:

- **Hero Section** (Line ~50-80):
  - Name: Update "Yash Sanap" with your name
  - Title: Modify the typing animation strings in `script.js`
  - Description: Update the hero description text
  - Social Links: Update URLs in the hero-social section

- **About Section** (Line ~120-160):
  - Update the about text with your personal story
  - Modify highlight items to match your strengths

### 2. Skills & Technologies

Update the skills section (Line ~170-220):
- Add/remove skill items in the skills-grid
- Update skill icons (Font Awesome classes)
- Modify skill categories as needed

### 3. Experience

Update the experience timeline (Line ~230-280):
- Add your work experience
- Update job titles, company names, and dates
- Modify job descriptions and tags
- Add or remove timeline items

### 4. Projects

Update the projects section (Line ~290-350):
- Add your projects
- Update project titles, descriptions, and tech stacks
- Add project links (GitHub, live demo)
- Modify project images/placeholders

### 5. Contact Information

Update the contact section (Line ~360-420):
- Update email address
- Update phone number
- Update location
- Update social media links

### 6. Colors & Theme

Customize colors in `styles.css` (Line ~5-30):
- Modify CSS variables in `:root` for light theme
- Modify CSS variables in `[data-theme="dark"]` for dark theme
- Update gradient colors
- Adjust accent colors

### 7. Typing Animation

Update typing strings in `script.js` (Line ~80-85):
```javascript
const typingStrings = [
    'Your Title 1',
    'Your Title 2',
    'Your Title 3',
    'Your Title 4'
];
```
